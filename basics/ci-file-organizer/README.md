# 📁 File Organizer Script

Automatically organizes a messy folder by sorting files into folders like Images, Documents, Videos, etc.

---

## 📌 Features

- 🧠 Uses `os` and `shutil` (no external libraries)
- 📁 Auto-creates folders if missing
- 🔁 Loops through all files
- 🔀 Moves files based on extension
- 🧹 Works great for Downloads folder cleanup

---

## 🚀 How to Run

```bash
python file_organizer.py
